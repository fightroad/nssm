# NSSM (Non-Sucking Service Manager) - Fork

本仓库是对原 NSSM 的 Fork，目标是在保持稳定性的前提下进行定制与增强。

## 项目简介
NSSM 是一个在 Windows 上将任意可执行程序以服务方式运行与管理的工具，支持启动/停止/重启、崩溃自动重启、标准输出/错误重定向、环境变量与工作目录设置等特性。本 Fork 计划在以下方向演进：
- 兼容性与稳定性增强
- 更完善的日志与诊断
- 可选的配置文件/环境变量扩展
- 构建脚本与自动化（如适用）
- 文档与示例完善

## 使用
- 安装服务：
  ```
  nssm install <ServiceName> <PathToExe> [args...]
  ```
- 移除服务：
  ```
  nssm remove <ServiceName>
  ```
- 更多参数与高级用法：参考原 NSSM 文档与本 Fork 的增补说明。
